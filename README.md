# wxPA

wxPA 是基于 [gewechaty](https://github.com/mikoshu/gewechaty)项目的二次开发。底层使用 [Gewechat](https://github.com/Devo919/Gewechat?tab=readme-ov-file)。

## 启动方式

### nodejs

```shell
node main.js
```

### docker-compose

目前此种启动方式还在实验中，gewe 直接打包，懒人的福音。

```shell
docker-compose up
```

## 功能

[ ] webUI
[ ] 定时任务
[ ] 延时任务
[ ] 群聊自动回复
[ ] 群聊自动加人
[ ] 群聊自动踢人
